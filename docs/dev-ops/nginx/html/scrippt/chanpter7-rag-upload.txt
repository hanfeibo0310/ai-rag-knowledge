### 标准化前端开发需求指令（AI工具精准识别版）
#### 一、核心目标
开发一款 **知识库文件上传前端页面**，支持与指定后端接口交互，实现知识库名称输入、多文件上传功能，要求界面美观、响应式适配、交互友好。

#### 二、技术栈限制（强制）
- 仅允许使用：HTML + 原生 JavaScript + Tailwind CSS
- 禁止使用：Vue、React、Angular 等任何前端框架，禁止额外引入非指定依赖（仅允许 Tailwind CSS + Font Awesome 图标库）

#### 三、后端接口规格（固定）
1. 接口地址：`http://localhost:8090/api/v1/rag/file/upload`
2. 请求方式：POST
3. 请求格式：multipart/form-data
4. 请求参数：
   - 必选参数：`ragTag`（字符串类型，知识库名称，不能为空）
   - 必选参数：`files`（多文件类型，仅支持 .md、.txt、.sql 格式）
5. 响应格式：JSON
   - 成功响应：`{"code":"0000","info":"调用成功"}`
   - 失败响应：`{"code":"非0000","info":"错误描述"}`
6. 跨域说明：后端已配置跨域（`allowCredentials="true"`，`origins="*"`），前端需适配跨域请求

#### 四、页面功能要求（详细）
1. 输入区域：
   - 知识库名称输入框（必填，占位提示："请输入知识库名称（如：产品文档、技术手册）"）
   - 表单验证：未输入知识库名称时，阻止提交并显示红色错误提示
2. 文件上传区域：
   - 支持两种上传方式：点击按钮选择文件 + 拖拽文件到指定区域上传
   - 文件过滤：自动过滤非 .md、.txt、.sql 格式的文件，无效文件不显示且给出提示
   - 文件列表展示：显示选中的文件名、文件大小（格式化显示：Bytes/KB/MB/GB），支持多文件选择
   - 列表交互：文件列表最大高度 40px，超出部分滚动显示
3. 上传交互：
   - 上传进度条：实时显示上传进度百分比（0%~100%）
   - 加载状态：上传中按钮禁用，显示加载动画（fa-spinner）
   - 提交按钮：全宽样式，视觉突出，文案："开始上传"（带上传图标 fa-upload）
4. 反馈提示：
   - 成功提示：绿色背景卡片，文案："文件上传成功！已加入知识库「{ragTag}」"，3秒自动隐藏
   - 错误提示：红色背景卡片，区分以下错误类型并精准提示：
     - 跨域错误："跨域错误！请检查后端跨域配置是否正确"
     - 超时错误："上传超时！（超时时间：2分钟）请检查网络或后端服务"
     - 接口错误："上传失败：{后端错误描述}"
     - 格式错误："后端响应格式错误！请检查接口是否返回合法JSON"
   - 提示样式：圆角卡片，高对比度配色，间距合理，易读性强

#### 五、界面设计要求（详细）
1. 视觉风格：
   - 主色调：蓝色（#3b82f6），辅助色：成功绿（#10b981）、错误红（#ef4444）、中性灰（#f8fafc、#1e293b）
   - 风格：现代简约、专业商务风，卡片式布局，带柔和阴影和圆角
2. 布局结构：
   - 头部导航：包含系统名称（AI 知识库管理系统）、导航菜单（文件上传、知识库列表、系统设置），固定在顶部
   - 主体区域：居中显示，最大宽度 3xl，内部包含输入区、上传区、进度条、提示区，卡片式设计（带阴影、圆角）
   - 页脚：版权信息（© 2025 AI 知识库管理系统 - 版权所有）、技术支持信息，深色背景，居中文本
3. 响应式适配：
   - 桌面端：控件间距适中，字体大小协调，最大宽度限制避免过宽
   - 移动端：自适应屏幕宽度，控件间距、字体大小适配小屏，无横向滚动
4. 交互细节：
   - 上传区域 hover 效果：边框变色（蓝色）、背景浅色高亮（#f0f9ff）
   - 按钮 hover 效果：阴影加深、背景色微暗（透明度 90%）
   - 输入框 focus 效果：蓝色边框+蓝色阴影（透明度 50%）

#### 六、技术参数要求（固定）
1. 超时时间：2分钟（120000毫秒）
2. 跨域请求配置：
   - 前端 fetch 配置：`mode: 'cors'`、`credentials: 'include'`
3. 兼容性：适配 Chrome、Firefox、Edge、Safari 主流浏览器
4. 性能优化：
   - 文件大小格式化处理
   - 内容可见性优化（content-visibility: auto）
   - 避免不必要的 DOM 操作

#### 七、禁止事项（强制）
1. 禁止使用任何前端框架（Vue、React、Angular 等）
2. 禁止引入额外 CSS 文件，所有样式通过 Tailwind CSS 实现（可自定义工具类）
3. 禁止支持 .md、.txt、.sql 以外的文件类型
4. 禁止添加无关功能（如下载、编辑、删除已上传文件等）
5. 禁止使用 jQuery 等第三方 JS 库，仅使用原生 JavaScript
